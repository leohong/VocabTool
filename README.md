# 🧠 單字極限特訓系統 V05 (強迫記憶版)
> 一款基於「主動回想」與「動態錯題演算法」的純前端單字記憶 Web 應用程式。

專為高強度的長期單字記憶計畫（如征服國中 2,000 單字、高中 7,000 單字）而設計。透過極簡的純前端架構，結合雙階段流水線訓練、嚴格的拼寫容錯機制，以及直覺的視覺化狀態燈號，幫助學習者打破遺忘曲線，將單字真正刻進長期記憶中。

## ✨ 核心特色 (Features)

* ⚡ **雙階段流水線訓練 (Two-Stage Pipeline)**
    * **第一關 [快速掃描]**：透過鍵盤方向鍵（←/→）或觸控滑動（Swipe），以直覺的閃卡模式快速篩選「認識」與「不熟」的單字。
    * **第二關 [強制盲測]**：第一關篩選完畢後，系統會強制隱藏英文，要求學習者憑藉肌肉記憶與大腦直覺「全手動拼寫」出完整單字。
* 🔥 **強迫記憶與嚴格糾錯 (Strict Punishment System)**
    * **手滑警告**：第一次拼寫錯誤視為手滑，給予警告不扣分。
    * **失憶判定**：第二次拼寫錯誤即判定為記憶斷裂，系統將觸發「強迫抄寫」機制，進度歸零並記錄於錯題本，徹底阻斷無效的瞎猜。
* 📊 **動態溫度計監控 (Thermometer Indicator)**
    * 隨時監控「錯題負債」數量，依據大腦記憶塞車程度給予綠（狀態良好）、黃（記憶塞車）、紅（過載警告）三種燈號提示。
    * 過載時可一鍵發動「降溫特訓（隨機 50 字大會考）」，透過「雙倍消除法」優先清空大腦的錯誤連結。
* 💾 **無後端、零部署 (Zero-Backend & Local Storage)**
    * 無需伺服器或資料庫，所有進度、字典與錯題資料皆安全地儲存於瀏覽器的 `localStorage` 中。支援中斷續傳，關閉網頁也不怕進度遺失。
* 📂 **多字庫管理與無痛匯入 (Multi-DB & Easy Import/Export)**
    * 支援建立多組獨立字庫（如 `vocab_2000`, `vocab_7000`）。
    * 提供完整的 JSON 備份/還原機制。
    * 支援純文字 (TXT) 向下相容的字典與錯題本匯入。

## 🚀 如何開始 (Getting Started)

本系統為單一 HTML 檔案架構 (Single File Application)，依賴 CDN 載入 React 與 Tailwind CSS，**無需任何安裝或建置步驟**。

1. 將本專案 clone 到本地端，或直接下載 `index.html`（或你命名的 html 檔）。
2. 在任何現代瀏覽器（Chrome, Edge, Safari, Firefox）中直接雙擊點開該 HTML 檔案。
3. 開始你的極限特訓！

## 📚 資料匯入格式 (Data Format)

你可以透過介面中的「匯入字典 (TXT)」功能，快速載入自己的單字庫。TXT 檔案的內容必須符合以下正則表達式格式：

**字典 TXT 格式範例：**
```text
=== 特訓完整字庫 ===

1. [n.] system --> 系統 (範例字)
2. [v.] acknowledge --> 承認/確認 (範例字)
3. [adj.] extreme --> 極端的

```
*(註：系統會自動抓取 [ ] 內的詞性、英文單字與 --> 後方的中文解釋。)*
**錯題本 TXT 匯入格式範例：**
```text
=== 核心錯題本個人紀錄 ===

1. 錯誤次數: 3次 | [n.] system --> 系統
2. 錯誤次數: 5次 | [adj.] extreme --> 極端的

```
## 🛠 技術棧 (Tech Stack)
 * **UI 框架**: React 18 (via CDN) + Babel (Standalone)
 * **樣式**: Tailwind CSS (via CDN)
 * **圖示**: 內建客製化 SVG Icons
 * **語音**: Web Speech API (原生瀏覽器 TTS)
 * **儲存**: HTML5 Web Storage API (localStorage)
## 💡 使用建議 (Usage Tips)
 1. **每日單字量**：建議依個人吸收能力，將左上角的「每日字數」設定在 30 ~ 100 之間。
 2. **關注燈號**：當介面亮起🟡黃燈或🔴紅燈時，請暫停推進新進度，優先點擊「🔥 降溫特訓」清除錯題本中的單字，避免記憶過載。
 3. **定期備份**：由於資料儲存於本地瀏覽器，清除瀏覽器快取可能會導致資料遺失。請務必養成定期點擊「JSON 全備份匯出」的習慣。
