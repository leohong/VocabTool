# 🧠 極限單字特訓系統 (VocabTool)

> 一款基於「主動回想」、「動態錯題懲罰」與「間隔重複」的純前端單字記憶 Web 應用程式。

🔗 **線上即用網址**：[https://leohong.github.io/VocabTool/](https://leohong.github.io/VocabTool/)  
📦 **內建字庫下載**：[國中 2000 單字庫](https://github.com/leohong/VocabTool/raw/main/2000_單字庫.txt) | [高中 7000 單字庫](https://github.com/leohong/VocabTool/raw/main/7000_單字庫.txt)

## 🚀 3 秒快速上手 (Getting Started)

本系統為單一 HTML 檔案架構 (Single Page Application)，支援線上免安裝即用，亦可下載至本地執行。

### 1. 開啟系統
* **線上即用 (推薦)**：造訪 **[線上特訓系統網址](https://leohong.github.io/VocabTool/)**。
* **本地雙擊開啟**：下載專案後直接雙擊開啟 `index.html` 即可直接使用。

### 2. 一鍵載入字庫與發動特訓
1. 在儀表板底部的「檔案管理與備份」區塊點擊 **`⬆️ 字` (匯入字典)** 按鈕，開啟 **「📥 選擇匯入來源」** 選單。
2. 點選 **🎒 載入內建 2000 單字庫**（或 🎓 **7000 單字庫**），系統將自動自 GitHub 雲端直載字庫。
3. 看到成功提示後，點擊上方 **「⚡ 發動今日特訓」** 即可立即開始特訓！

## 🎮 核心功能指南 (Core Features)

### 1. 指揮中心
* **⚡ 發動今日特訓**：開始當天設定的每日新單字特訓。特訓中會自動安插今日到期的歷史「幽靈單字 👻」進行突襲。
* **🔥 降溫：錯題大會考**：從當前錯題集中隨機抽取最多 50 個字進行純拼寫盲測，幫助快速消除錯題債務。
* **🏛️ 深度：歷史隨機抽查**：隨機抽取最多 50 個已封存於歷史殿堂的單字進行純拼寫抽驗，主動檢測長期記憶。
* **🎧 聽讀：聽音背單字**：進入聽音背單字特訓播放器。支援播放自訂範圍的單字、拼讀字母、朗讀中文釋義與例句，並配備眼睛隱藏遮蔽（盲聽）功能。
* **🔍 預覽今日特訓新單字**：在發動特訓前，可以先預覽今日的單字清單與發音。
* **🗂️ 預覽全字庫單字**：開啟全字庫預覽面板，提供關鍵字即時篩選、分頁瀏覽、編輯與徹底刪除功能。
* **📖 查字典與新增單字**：整合線上英語字典與翻譯 API，查詢後可手動新增至字庫中。

### 2. 檔案管理與 4 種匯入/備份機制
點擊儀表板底部的檔案管理按鈕，系統提供彈性完整的備份與匯入功能：
* **`⬆️ 字` (選單式匯入單字庫)**：開啟「📥 選擇匯入來源」選單，支援：
  1. 🎒 **載入內建 2000 單字庫**（雲端直載）
  2. 🎓 **載入內建 7000 單字庫**（雲端直載）
  3. 📁 **從本機選擇檔案 (.txt)**（選取自訂單字檔）
* **`⬇️ 字` (匯出單字庫)**：將當前字庫匯出為標準 TXT 文字檔。
* **`⬆️ 殿` / `⬇️ 殿` (歷史殿堂備份/還原)**：匯入或匯出歷史殿堂個人紀錄 TXT 檔。
* **`⬆️ J` / `⬇️ J` (JSON 系統全備份)**：一鍵還原或導出包含所有字庫單字、學習進度、設定及歷史紀錄的極限完整備份檔 (`.json`)。

## ⌨️ 鍵盤快捷鍵與觸控手勢

### ⌨️ 鍵盤快捷鍵 (電腦桌面)
* **第一關 [快速篩選]**：
  * 按 **`←` 方向鍵**：歸類為 **「不熟 / 不認識」**。
  * 按 **`→` 方向鍵**：歸類為 **「認識 / 記住了」**。
* **第二關 [強制盲測]**：
  * 輸入完畢後按 **`Enter` 鍵** 送出檢查。
  * 若觸發手滑警告或強迫重抄，再次輸入完畢後按 **`Enter` 鍵** 進入下一個字。

### 📱 行動裝置觸控手勢
* **第一關 [快速篩選]**：
  * 在單字卡區域 **向左滑動 (Swipe Left)**：歸類為 **「不熟」**。
  * 在單字卡區域 **向右滑動 (Swipe Right)**：歸類為 **「認識」**。

## 📚 資料匯入格式規範 (Data Formats)

### 1. 字典 TXT 格式範例：
必須包含 `=== 特訓完整字庫 ===` 開頭，單字格式為 `編號. [詞性] 英文 --> 中文`，可用 `||` 追加例句：

```text
=== 特訓完整字庫 ===

1. [n.] system --> 系統 || A computer system is complex. (電腦系統很複雜。)
2. [v.] acknowledge --> 承認/確認 || I acknowledge my mistake. (我承認我的錯誤。)
3. [adj.] extreme --> 極端的
```

> [!TIP]
> **聽讀靜音優化**：例句格式建議採用 `英文例句 (中文翻譯)`，系統在聽讀朗讀例句時，會自動過濾靜音中文括號部分，維持盲聽專注度。

### 2. 歷史殿堂 TXT 格式範例：
必須包含 `=== 歷史殿堂單字個人紀錄 ===` 開頭，格式為 `編號. 錯誤次數: X次 | [詞性] 英文 --> 中文`：

```text
=== 歷史殿堂單字個人紀錄 ===

1. 錯誤次數: 3次 | [n.] system --> 系統 || A computer system is complex. (電腦系統很複雜。)
2. 錯誤次數: 5次 | [adj.] extreme --> 極端的
```

### 3. JSON 格式單字清單 / 系統完整備份：
* **自訂單字清單 (Array)**：
  ```json
  [
    { "en": "apple", "pos": "n.", "zh": "蘋果", "eg": "An apple a day keeps the doctor away." },
    { "en": "challenge", "pos": "n.", "zh": "挑戰" }
  ]
  ```
* **系統極限完整備份檔 (v2.0 `full_system`)**：透過 `⬇️ J` 導出的 JSON 檔，可一次性完整還原多字庫與學習歷程。

## 🛠 技術棧 (Tech Stack)

* **UI 框架**：React 18 (via CDN) + Babel (Standalone)
* **樣式**：Tailwind CSS (via CDN)
* **圖示**：內建客製化 SVG Icons
* **語音**：Web Speech API (原生瀏覽器 TTS，美式英語語音)
* **儲存**：HTML5 Web Storage API (localStorage)
* **測試**：Python + Selenium Headless Chrome E2E 自動化測試
